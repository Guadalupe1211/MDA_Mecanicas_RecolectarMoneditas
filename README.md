## Descripción

Este proyecto consiste en un minijuego de recolección desarrollado en Unity con el objetivo de aplicar el marco de diseño de videojuegos 
**MDA (Mechanics–Dynamics–Aesthetics)**. El jugador controla un personaje que debe desplazarse por el escenario para recoger monedas, ganar experiencia 
y subir de nivel.

El proyecto busca demostrar cómo mecánicas simples pueden generar dinámicas de exploración y progresión, produciendo una experiencia satisfactoria 
para el jugador mediante recompensas constantes y objetivos claros.

## Objetivos del Proyecto

* Aplicar el modelo MDA al diseño de videojuegos.
* Implementar mecánicas básicas de movimiento y recolección.
* Diseñar un sistema simple de experiencia y niveles.
* Analizar la relación entre mecánicas, dinámicas y estéticas.
* Desarrollar un prototipo funcional en Unity.

## Mecánicas Implementadas

### 🎮 Movimiento

El jugador puede desplazarse libremente por el escenario utilizando los controles asignados.

**Función:**

* Explorar el mapa.
* Alcanzar monedas y recompensas.
* Interactuar con los elementos del entorno.

### 🪙 Recolección de Monedas

Las monedas se encuentran distribuidas por el escenario y pueden ser recolectadas al entrar en contacto con ellas.

**Función:**

* Incrementar la cantidad de monedas obtenidas.
* Recompensar la exploración del jugador.
* Generar retroalimentación inmediata.

### ⭐ Sistema de Experiencia y Niveles

Cada moneda recolectada otorga experiencia.

**Función:**

* Incrementar el progreso del jugador.
* Permitir subir de nivel.
* Motivar la continuidad de la partida.

## Aplicación del Modelo MDA

### Mechanics (Mecánicas)

* Movimiento del personaje.
* Recolección de monedas.
* Obtención de experiencia.
* Sistema de niveles.

### Dynamics (Dinámicas)

Las mecánicas implementadas generan:

* Exploración del escenario.
* Búsqueda activa de recompensas.
* Acumulación de recursos.
* Progresión continua mediante experiencia.
* Motivación para recolectar más monedas.

### Aesthetics (Experiencia del Jugador)

Las dinámicas producen sensaciones de:

* Libertad de movimiento.
* Curiosidad por explorar.
* Satisfacción al recolectar objetos.
* Sensación de progreso.
* Motivación por alcanzar nuevos niveles.
* Logro al completar objetivos.


## Tecnologías Utilizadas

* Unity
* C#
* Unity UI
* Sistema de Colisiones
* Rigidbody
* Trigger Events


## Aprendizajes Obtenidos

Durante el desarrollo de este proyecto se analizaron las diferencias entre las mecánicas utilizadas en videojuegos Free-to-Play y Premium/AAA, 
identificando cómo sistemas simples de progresión y recompensas pueden aumentar el compromiso del jugador.

Asimismo, se aplicó el modelo MDA para comprender cómo las decisiones de diseño afectan la experiencia final del usuario, 
relacionando directamente las mecánicas implementadas con las emociones y sensaciones generadas durante el juego.
